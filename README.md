# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered career guidance platform that helps students improve their resumes, prepare for interviews, analyze job matches, and receive personalized career guidance using Google's Gemini AI.

---

## ✨ Features

- 🤖 AI Career Chat
- 📄 Resume Analyzer
- 🎯 ATS Job Match
- 🎤 AI Interview Coach
- 📊 Dashboard
- 💻 Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API

### Database
- MongoDB (Integration in Progress)

## ⚙️ Installation

### Clone Repository

### Backend

cd Backend
npm install
npm run dev
